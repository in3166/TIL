# 일급
- `값`으로 다룰 수 있다.
- `변수`에 담을 수 있다.
- 함수의 `인자`로 사용될 수 있다.
- 함수의 `결과`로 사용될 수 있다.
```js
const a = 10; // 변수에 값을 담는다.
const add10 = a => a + 10; // add10 함수의 인자로 a가 전달될 수 있다., 함수의 결과는 'a+10'이라는 값으로 사용될 수있다.
add10(10); // 
```
